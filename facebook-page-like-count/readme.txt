== Description ==

Facebook Page Like Count is a simple plugin that allows you to display the total like count of any Facebook page. The Like Count can be added via shortcode or template tag to your theme. 

<strong>Additional Features:</strong>
<ul>
<li>Ability to auto update the like count every hour, once daily, or twice daily</li>
</ul>

[__For more information or to request additional features, please visit the plugin page__](http://presshive.com/plugins/display-your-facebook-page-like-count/)


== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.
     
2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Visit the plugin admin page via Settings -> Facebook Page Like Count

= To Display Facebook Page Like Count Via Shortcode = 

* To include the Facebook Page Like Count in your post or pages use `[flc_count]`

= To Display Facebook Page Like Count Via Template Tag = 
* Use
<code>
<?php flc_count( $echo = true ); ?>
</code>
